Currently, banana supports the following named entities:

- `&amp;`: `&`
- `&gt;`: `>`
- `&lb;`: `{`
- `&lt;`: `<`
- `&sp;`: ``

Banana also supports numerical entities (eg `&#32;` means space (character code
32)). Currently, only decimal numbers are supported as the number
